üí° Photoshop Pro Tips

  • Always export at 2x resolution for retina displays (multiply dimensions by 2)
  • Use object-fit: cover means images will crop to fill - center important content
  • Face-focused images work best (center-top cropping)
  • Maintain 72 DPI for web optimization
  • Export as JPEG (80-90% quality) for best file size vs quality
  • Use PNG only for images with transparency
  • Keep file sizes under 500KB for fast loading
  • Test images in different card positions before finalizing
